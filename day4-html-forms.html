What I've Learned About HTML Forms

14 July, 2025

I've been diving into HTML forms again - both as a refresher for this blog and also to gain a deeper understanding before I move forward. And honestly, there's so much more to them than I intially thought. What started as a simple way to collect user input has turned into this fascinating study of user experience, accessibility, and web standards.

My Journey with Form Basics

When I first started working with forms, I thought they were just about throwing some input fields into a <form> tag and calling it a day. Boy, was I so wrong! I learned that every form needs two crucial attributes that I didn't quite use to fully understand:

  • action: Tells the browser where to send the data
  • method: Determines how the data gets there
<form action="submit.html" method="GET">
  <!-- All form information goes here -->
</form>

I used to just copy-paste this without really understanding what POST and GET means, but now I get it! (Thanks to the video tutorial by Web Dev Simplified.)

  • POST: Useful for when you have a server where you need to save some information
  • GET: Sends the collected information to another page on your site

The Input Types That Changed Everything

When I used to fill in forms online, I used to think that they were all just text field boxes. But now that I've learned about HTML forms, I've come to realized that it's more than that.


<input type="email" name="email" placeholder="your@gmail.com">

The email input automatically valids email format, which would require the person to enter a valid email address before the form can be submitted.


<input type="password" name="password">

This is a password input, which hides the characters you type in the field box.


<input type="tel" name="phone">

The tel input requests for you to enter your phone number, and it also brings up the number and automatically on mobile devices.

These small details make such a huge difference in the user experience, and I'm so glad that I know about them now!

There are also other input types that feel like Christmas mornings. Date pickers, color selectors, rang sliders - all built right into the browser!
