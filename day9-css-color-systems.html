CSS Color Systems: What I've Learned So Far

19 July, 2025

I've been refreshing my knowledge on CSS lately to post for this learning journey blog, and I realize that one thing that kept tripping me up was all the different ways to specify colors. Coming from a design background where I mostly picked colors from palettes, the technical side of CSS colors felt overwhelming. But after watching tutorials and playing with them for awhile, I'm starting to see how each system has its own strengths.

Here's what I've figured out about the main color systems in CSS, along with some things I wish I'd known earlier.

RGB: The Building Blocks

RGB was the first color system I really understood. It's basically mixing red, green, and blue light, with each value going from 0 to 255. Think of it like having 3 dimmer switches.

.myelement {
  color: rgb(255, 0, 0); /* pure red */
  color: rgb(0, 255, 0); /* pure green */
  color: (0, 0, 255); /* pure blue */
   color: rgb(128, 128, 128); /* gray */
}

What I like about RGB is that it's pretty intuitive once you get the hang of it. Higher numbers = more of that color. But to be frank, we all use some color picker now and just copy paste the values, isn't it?

Hexadecimal: The Shorthand

Hex codes are just RGB in disguise, but written in hexadecimal (base 16). I used to be intimidated by them, but they're actually quite logical.

.my-element {
  color: #ff0000; /* same as rgb(255, 0, 0) */
  color: #00ff00; /* same as rgb(0, 255, 0) */
  color: #0000ff; /* same as rgb(0, 0, 255) */
  color: #808080; /* same as rgb(128, 128, 128) */
}

I usually reach for hex codes because it's what I'm used to when I was a designer, but also because they're compact (which means the code would look neater)

HSL: The Intuitive One

HSL (Hue, Saturation, Lightness) was a game-changer for me. It matches how I actually think about colors.

.myelement {
  color: hsl(0, 100%, 50%); /* pure red */
  color: hsl(120, 100%, 50%); /* pure green */
  color: hsl(240, 100%, 50%); /* pure blue */
  color: hsl(0, 0%, 50%); /* gray */
}
  • Hue is the color itself (0-360 degrees around a color wheel)
  • Saturation is how vibrant it is (0%=gray, 100%=full color)
  • Lightness is how bright it is (0%=black, 100%=white)

Adding Transparency: RGBA and HSLA

Both RGB and HSL have alpha versions that let you control transparency. The alpha value goes from 0 (completely transparent) to 1 (completely opaque).

.transparent-elements {
  background: rgba (255, 0, 0, 0.5); /* 50% transparent red */
  background: hsla(0, 100%, 50%, 0.3);
}

Gradients: Where It Gets Fun

Gradients were intimidating at first, but they're actually just transitions between colors. The most common types are linear and radial.

.gradient-examples {
  /* simple linear gradient
  background: linear-gradient(to right, #ff0000, #0000ff);

  /* angled gradient */
  background: linear-gradient(45deg, #ff0000, #0000ff);

  /* multiple colors */
  background: linear-gradient(to bottom, #ff0000, #00ff00, #0000ff);

  /* radial gradient */
  background: radial-gradient(circle, #ff0000, #0000ff);
}
