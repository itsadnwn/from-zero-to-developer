My CSS Flexbox Learning Guide

21 July, 2025

I did not have the best experience learning about CSS Flexbox. It took me awhile to understand what's going on, and what helped me was Colt Steele's video!

Making a Container into a Flex Container

The first thing I learned is that you need to explicitly tell CSS that you want to use flexbox. You do this by adding display: flex; to your container:

.my-container {
  display: flex;
}

That's it! Once I added this, my container becomes a "flex container" and all its direction child become "flex items".

What Goes Inside a Flex Container?

This was something that confused me at first. The flex container can hold any HTML elements as direct children. The keyword here is direct children - only the immediate child elements become flex items. If there are nested elements, those won't be affected by the flex properties.

The Five Main Flex Container Properties

flex-direction

This property controls which direction your flex items flow. I think of it as setting the "main axis" of the container.
