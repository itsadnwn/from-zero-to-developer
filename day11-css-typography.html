From Designer to Developer: My Journey with CSS Typography and Google Fonts

20 July, 2025

As someone who transitioned from design to development, I thought I had typography figured out. After years of working with fonts in design software, how hard could web typography be? Well let me tell you - it was a humbling experience that taught me just how different web typography is from print and design work.

The Reality Check: Design Skills Don't Directly Translate

Coming from a design background, I was quite confident about typography. I knew about kerning, leading, font pairing, and hierarchical systems. I understood the emotion impact of typefaces and had strong opinions about Comic Sans (let's admit it, we all do).

But when I started coding, I quickly realized that web typography operates under completely different constraints:

  1. Performance matters iin ways I never considered in design
  2. Accessibility requires technical implementation, not just good intentions
  3. Responsive design means your carefully crafted typography needs to work across infinite screen sizes
  4. Loading states and fallback fonts aren't concerns in static design work

Translating Design Typography Knowledge to CSS

In design software, you pick a font and it works. In CSS, you need to think about fallbacks:

/* My designer brain wanted this */
.heading {
  font-family: 'Playfair Display';
}

/* But the web requires this */
.heading {
  font-family: 'Playfair Display', 'Times New Roman', Georgia, serif;
}

This was my first lesson in defensive coding. You have to assume things will fail and plan accordingly.

Linking Google Fonts to HTML & CSS

Method 1: HTML Link Tag (Recommended)

Add this to your HTML <head> section:

<head>
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;600;700&display=swap" rel="stylesheet">
</head>

Then use this in your CSS:

body {
  font-family: 'Open Sans', sans-serif;
}

Method 2: CSS @import

Add this at the very top of your CSS file:

@import url('https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;600;700&display=swap');

body {
  font-family: 'Open Sans', sans-serif;
}

How to Get the Font URL:

  1. Go to Google Fonts
  2. Search for your desired font
  3. Add the desired font to cart
  4. Click on '<> Get embed code'
  5. Select the style and weight variable if applicable
  6. The code will be ready on the right side of the page
